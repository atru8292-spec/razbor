# Razbor — дизайн-ориентир

B2B-инструмент для владельцев сайтов. Должен выглядеть дорого, экспертно, спокойно.
Принцип: сдержанный editorial, а не громкий SMM. Доверие важнее «вау».

## Тональный север
Приглушённо, премиально, много воздуха, объект/данные на чистой бумаге, уверенная типографика
без крика. Ориентир — сдержанный продуктовый editorial, НЕ подростковый гранж и НЕ мягкий тг-блог.

## Палитра (locked)
- `#4E0000` оксблад — брендовый primary + severity HIGH (где сайт теряет деньги)
- `#362017` эспрессо / почти чёрный — основной текст
- `#FAF5E7` кремовая бумага — фон страницы
- `#A9BBD1` пыльно-голубой — спокойный акцент, severity OK / нейтральный UI
- `#263650` тёмно-синий — глубина, шапки, severity MEDIUM, чрома интерфейса

Severity-система выдачи строится прямо из палитры:
high = оксблад, medium = навахо/синий, low = приглушённый, ok = пыльно-голубой.

## Типографика
- Заголовки: Unbounded (капсом для логотипа RAZBOR и крупных хедов)
- Текст: Manrope
- Чисто, без декоративных и гранж-шрифтов. Дисплейные шрифты из моих SMM-наборов сюда НЕ тащить.

## Что брать из рефов
- Сдержанность и «дорого» из премиального продуктового поста (объект на бумаге, воздух).
- Editorial-сетку: жёсткая типо-иерархия, мелкие лейблы-теги, высокий контраст, детали (штрихкод-подобные акценты).

## Чего НЕ делать (anti-slop + B2B-фильтр)
- Ярко-красный (у нас оксблад), фиолетовые градиенты, три одинаковые карточки, дефолтный Inter.
- Гранж и рукописные шрифты, кричащие капсы с маркером — подрывают доверие.
- Скруглённый «миленький» SMM-стиль с пилюлями и стикерами-курсорами — читается как тг-блог, не как сервис.
- Стоковые улыбки. Визуал показывает данные и результат.

## Применять
Навык frontend-design + фильтр против AI-слопа. Это документ для Шага 7 (премиальная выдача,
экран ожидания, лендинг) и для OG-карточки (satori + resvg).
