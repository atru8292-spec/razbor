в воронку дожима → цель закрыть на редизайн (услуга от 20к).

brutalism.

## ДЕПЛОЙ (Арина делает сама в обычном терминале)
```
ssh root@31.77.197.158
screen -S deploy   # SSH рвётся на долгом build; восстановить: screen -r deploy
cd /var/www/razbor && git checkout package-lock.json && git pull && npm install && npm run build && pm2 restart all --update-env
```
Только фронт → pm2 restart razbor-web --update-env. Бот/воркер → pm2 restart
razbor-bot razbor-worker. Билд 5-10 мин. --update-env чтобы pm2 подхватил .env.

нужен ребилд.

## ЧТО УЖЕ СДЕЛАНО (всё в проде)
- Сервис AI-аудита от инфры до прода (скрапер, воркер, бот, база).
- Редизайн выдачи (brutalism, аккордеоны, AEO-блок).
- Воронка по лестнице Ханта (бот + почта, кнопки активного пути).
- Аналитический кабинет /admin: честная воронка на униках, AI-маркетолог
  (кнопка «разобрать», gpt-5.4-mini, ~0.18¢), мини-CRM (карточка лида: аудит +
  статус + переписка бота как мессенджер), источники трафика UTM, реалтайм,
  статистика бота. Плотная раскладка (инвертированная пирамида).
- Короткий гейт: одно поле почта + согласие. Подарок (чек-лист) только в боте
  (завлечение). Тг получаем автоматом при переходе в бота. Двойной охват.
- Бот: кнопки активного пути (🔍 главная проблема / 📋 что чинить / 💬 разобрать
  лично), синхронизация с расписанием через followup_stage. Переписка в карточке.
- QA качества разборов (5 типов сайтов): жаргон убран, названия направлений
  человеческие, оценки откалиброваны (контраст), аккордеон раскрыт.
- Самоаудит getrazbor.ru пройден (нашёл реальный баг — дубль кнопки, починен).
- SEO/AEO фундамент: robots (пускает AI-боты), sitemap, JSON-LD (Org/WebSite/
  SoftwareApplication/FAQPage), llms.txt, FAQ под боль, мета с языком боли.
  Подтверждён в Яндекс.Вебмастер + Google Search Console, sitemap добавлен.
- Функция «Как это починить →» (демо-переделка первого экрана): по кнопке,
  gpt-5.4 vision берёт скрин хиро из готового аудита → новый HTML/CSS + объяснения
  было/стало/почему. Превью в sandboxed iframe, кэш, ~4.5-5.9¢. Откалибрована на
  3 нишах (заголовки живые, дизайн под нишу, мобайл-адаптив, без слопа, цену не
  выдумывает). Поднята наверх выдачи (после главной утечки).
- Пересмотр выдачи: ширина десктопа (вердикт шире, не кишка), блок нейропоиска
  со вступлением, лоадер без «думаю как дизайнер».
